# Analyzing LinkedIn Job Market

This project is about analyzing industry trends in the LinkedIn job market with an application of Latent Dirichlet Allocation (LDA) for topic modeling on job descriptions, followed by dimensionality reduction using t-SNE and UMAP to visualize semantic clusters.
